Special Topics in Design - I (DSL 810)

I would be offering an elective course in spring semester 2020 on Advanced Prototyping. Since this course is not there in the course catalog yet, so I am sharing the contents of this course. It would be called Special Topics in Design-I (DSL 810).

Course Contents:

The focus of this course would be getting hands-on with electronics and software. The students would learn about web development, programming languages, microcontroller programming, input devices including sensors, output devices including actuators, networking and communication with Bluetooth and Wi-Fi modules, user interface (UI) and application programming to develop mobile apps and desktop interfaces, PCB design and fabrication. Introduction to data science and AI/ML for designers is planned. Along these lines, we would also understand how to analyze Big Data generated from experiments using sensors. Course project would utilize the skills learnt in the class.

This course should be useful both for students interested in interaction and product design. There are no prerequisites. Please share this info with your friends as this course is not in the course catalog yet. Please also let me know for questions.
